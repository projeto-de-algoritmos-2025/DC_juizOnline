## Instalação

**Linguagem:** Python 3.10+

### Pré-requisitos


- Para Python:
  - Python 3 instalado
  - `pip` (gerenciador de pacotes do Python)

- Para C++:
  - Compilador C++ (como `g++`) instalado
  - Recomendado: usar um ambiente com suporte a C++17 ou superior
